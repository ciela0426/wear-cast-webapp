# ✨ Feature

- IP의 위치 정보에 따른 날씨 정보(상태, 평균기온, 습도, 풍속 등)
- 기온별 옷차림
- 대기질에 따른 마스크 유/무
- 우산 필요 유/무
- 적정 온습도
- 내일/모레 날씨 3시간 단위
